# 📅 Story Spark Calendar

A modern, interactive calendar application built with React, TypeScript, and Tailwind CSS. It combines smooth animations, smart date selection, and note-taking features into a visually engaging dashboard experience.

---

## 🚀 Features

### 🎨 UI & Animations

* Animated hero header with dynamic month-based gradients
* Smooth transitions using **Framer Motion**
* Floating particle effects and interactive visuals
* 3D flip animation during month navigation

### 📆 Smart Calendar

* Interactive calendar grid
* Click & drag (or touch) to select date ranges
* Highlights:

  * Current day
  * Weekends
  * Selected ranges

### 📝 Notes & Goals

* Add notes or goals for selected dates
* Data persists using **localStorage**
* Simple and intuitive form handling

### 🎉 Holiday Indicators

* Highlighted holidays with visual markers
* Tooltip and badge animations
* Dedicated legend for reference

### 📱 Responsive Design

* Fully mobile-friendly
* Touch gesture support for date selection
* Adaptive layout for different screen sizes

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/anjihan3601K/Calender_Challenge.git
cd Calender_Challenge
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run development server

```bash
npm run dev
```

👉 Open: http://localhost:5173

---

## 🏗️ Production Build

```bash
npm run build
npm run preview
```
